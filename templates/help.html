This page contains explanatory information about the main website features, which can be seen in the vertical menu bar.

Home
Leads to the homepage of the website.

New Transaction
Leads to a page where user can create a new transaction. Type the recipient node's address (text) in the first input box and the amount of coins you wish to send (integer) on the second input box, both inputs are required. Then, by pressing the button "Create Transaction", a new transaction is created and the specified amount of coins will be sent to the specified recipient node.
If the transaction was created successfully the message "The transaction was conducted successfully" will appear, otherwise, if an error occured, an appropriate message will appear that will guide the user to correct their mistake.

View
Leads to a page which displays all the transactions contained in the last valid block of the current blockchain. Some information about the last valid block (like its hash) are also displayed in this page.
If there is no block in the blockchain an appropriate message appears.

Balance
Leads to a page where the balance of the current node's wallet is displayed.
